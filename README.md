# Minecraft PE Switch Port

A native Nintendo Switch port of Minecraft Pocket Edition (v0.6.1 alpha compatible).

## Controls

| Button | Action |
|--------|--------|
| A | Jump / UI Confirm |
| B | Sneak / UI Back |
| X | Debug/Perspective toggle |
| Y | Drop item |
| Minus (-) | Open inventory / Block selection |
| Plus (+) | Pause / Back |
| L3 | Sneak |
| ZL | Hotbar scroll left |
| ZR | Hotbar scroll right |

## Features

- Full survival and creative mode gameplay
- World generation and saving
- Multiplayer support (LAN discovery and Direct Connect **COMING SOON**)
- Sound system with all original game sounds
- Touchscreen support for UI navigation
- Custom buttons/options
- Performance optimizations for Switch hardware

## Multiplayer COMING SOON

The game supports two ways to join multiplayer games:

1. **LAN Discovery**: Select "Join Game" from the main menu to scan for local servers on port 19132
2. **Direct Connect**: Select "Direct Connect" to enter a custom IP address and port

## Installation

Download the latest `Minecraft_PE.nro` from the Releases section and place it on your Switch SD card.

## Requirements

- Nintendo Switch with custom firmware
